## Overview

Task manager web app was specifically designed to test web broser' Local Storage which functions as a permanent storage for users tasks. Some of the features used on the project were:

- **JavaScript**.
- **HTML**.
- **DOM**.
- **querySelector**.
- **Bootstrap**.
